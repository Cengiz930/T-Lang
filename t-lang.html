Temel Komutlar

İşte T-Lang'in temel 10 komutu:

ekrana_yazdır <metin>
Ekrana belirtilen metni yazdırır.
Örnek: ekrana_yazdır "Merhaba Dünya"
değişken <ad> = <değer>
Yeni bir değişken oluşturur.
Örnek: değişken isim = "Ahmet"
eğer <koşul> { ... }
Koşul sağlanıyorsa belirtilen işlemleri yapar.
Örnek: eğer yaş >= 18 { ekrana_yazdır "Reşitsiniz" }
döngü <sayı> { ... }
Belirtilen sayı kadar döngü çalıştırır.
Örnek: döngü 5 { ekrana_yazdır "Merhaba" }
fonksiyon <ad> { ... }
Yeni bir fonksiyon tanımlar.
Örnek: fonksiyon selamla { ekrana_yazdır "Merhaba!" }
çalıştır <fonksiyon_adı>
Tanımlı bir fonksiyonu çalıştırır.
Örnek: çalıştır selamla
liste <ad> = [değer1, değer2, ...]
Yeni bir liste oluşturur.
Örnek: liste meyveler = ["elma", "armut", "muz"]
girdi_al <değişken_adı>
Kullanıcıdan girdi alır ve belirtilen değişkene atar.
Örnek: girdi_al kullanici_adi
döngü_iken <koşul> { ... }
Koşul sağlandığı sürece döngüyü çalıştırır.
Örnek: döngü_iken sayi < 10 { ekrana_yazdır sayi }
yardım
Tüm komutları listeler.
Örnek: yardım

Nasıl Kullanılır?

T-Lang'i kullanmak çok kolay! Yukarıdaki terminale komutları yazıp Enter tuşuna basmanız yeterli. Örneğin:

ekrana_yazdır "Programlamaya hoş geldiniz!"

Komutlar tamamen Türkçe olduğu için anlaması ve kullanması çok kolay.
